# Primeiro acesso

- Para seu primeiro acesso siga os seguintes passos.

  1. Abra sua pasta de Projetos da Unity (criada em aula)
  2. Dentro dela abra o GIT BASH Here
  3. Escreva esse comando dentro dela:
```
git clone
```
